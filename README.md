# SemAssist - Student Semester Assistant

A comprehensive web application designed to help students organize and manage their academic life.

## Features
- **Dashboard**: Get an overview of your semester at a glance
- **Assignments**: Track and manage your assignments with deadlines
- **Timetable**: Organize and view your course schedule
- **Preferences**: Customize your settings and preferences
- **Responsive Design**: Works on desktop and mobile devices

## Tech Stack
- **Frontend**: React with Vite, React Router, Axios
- **Backend**: Node.js with Express, MongoDB with Mongoose
- **Styling**: CSS with Lucide React icons
